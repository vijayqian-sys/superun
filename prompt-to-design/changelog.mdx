This page lists recent updates, new features, and improvements to the Prompt.to.design plugin.  
Updates are listed in reverse chronological order.

---

## Version 1.3.0 - December 2025

### Added
- Introduced **Template Remix** for generating new layouts from 1000+ templates using prompt-based customization.
- Added **Frame Redesign** to instantly restyle any selected frame into a new visual theme (Minimalist, Dark Mode, Glassmorphism, etc.).
- Released **Real Prototype**, enabling automatic conversion of static frames into fully interactive, clickable prototypes with container detection and navigation logic.

### Improved
- Enhanced layer parsing and layout detection to support more complex template structures in Template Remix.
- Improved visual recognition for Frame Redesign to better preserve layout hierarchy while applying new styles.
- Updated prototype rendering engine for Real Prototype, resulting in smoother transitions and faster interaction generation.

### Fixed
- Resolved an issue where certain uploaded reference images could fail to load in Template Remix.
- Fixed minor alignment inconsistencies when inserting redesigned frames back into Figma.
- Corrected a bug causing prototype previews to render incorrectly on high-resolution displays.

### Notes
- Template Remix now supports both page-level templates and component-level templates for broader coverage.
- Real Prototype supports auto-containerization and basic action mapping; advanced flows will roll out in upcoming versions.

## Version 1.2.0 - November 2025

### Added
- Introduced **Smart Fill** feature for context-aware content generation inside frames.  
- Added **pricing page** integration with new Free Trial and Plus Plan models.  
- Added **FAQ** and **Getting Started** sections in documentation.

### Improved
- Enhanced performance in **Style Transfer**, reducing generation latency by 35%.  
- Improved layout detection for **Region Design** to better preserve grid alignment.  
- Updated in-app onboarding text and placeholders for clarity.

### Fixed
- Resolved a bug where large image uploads could cause preview errors.  
- Fixed inconsistencies in credit usage display between sessions.

### Notes
- Early access users are automatically upgraded to Plus Plan until the next billing cycle.  
- For developers using custom component libraries, Figma integration setup now requires reauthorization.
