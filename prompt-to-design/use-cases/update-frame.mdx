title: "Update or refactor an existing design"
---


## Scenario
You want to modernize an old UI but keep its structure.

## Challenge
Refactoring UI by hand is slow and error-prone.

## How prompt.to.design helps

- Select the old **Figma Frame** you want to refresh  
- Enter a modernization prompt, such as:  
  **“Modernize this settings page while keeping the same layout structure. Use cleaner typography, updated form components, improved spacing, and a more polished sidebar design.”**  
- Click **Generate** to apply a refreshed, modern style
