superun 具有原生无缝集成,可以导出代码并添加后端.但如果您想为应用程序添加特定功能,也可以集成外部 API.

## 什么是 API?

API 是软件应用程序相互通信和共享数据的一种方式.不和的 API 提供不和的功能,例如发送电子邮件或处理付款.您可以在我们博客上的这篇文章中详细了解 API 的强大功能.
