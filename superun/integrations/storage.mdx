## Overview

superun Storage is a file storage and management service that allows you to securely store, organize, and deliver files for your applications. With superun Storage, you can easily manage your assets through an intuitive visual interface and deliver them globally.

## Accessing superun Storage

You can access superun Storage from **Build → Services → superun Storage** in your project settings.

<img src="https://b.ux-cdn.com/uxarts/files/t20251224144550/4yt9dxrl.png" alt="superun Storage Asset Management" />


## Features

### Asset Management

superun Storage provides powerful asset management capabilities to help you organize and manage your files efficiently.

#### Visual File Browser

Upload, move, and organize files through an intuitive drag-and-drop interface. No need for complex command-line tools or third-party clients. The visual file browser makes it easy to manage your assets directly within superun.

<img src="https://b.ux-cdn.com/uxarts/files/t20251224144550/9v1inffc.png" alt="superun Storage Visual File Browser" />

When you click a file in the list, a details panel opens on the right where you can **download the file** or **copy its URL** for use in your app.

<img src="https://b.ux-cdn.com/uxarts/files/t20251224170645/rd7nkh06.png" alt="superun Storage file details panel with download and copy URL actions" />

You can also click the **More** button on the right side of a file row to quickly **download** or **delete** that file.

<img src="https://b.ux-cdn.com/uxarts/files/t20251224170807/vkitpaev.png" alt="superun Storage file row more actions with download and delete" />

#### Automatic Metadata Capture

superun Storage automatically identifies file types, sizes, and dimensions, making it easy to filter and search through thousands of assets. This metadata is captured automatically, so you can quickly find the files you need without manual tagging.

### Global Delivery

superun Storage is designed to deliver your files quickly and cost-effectively to users around the world.

#### Edge Acceleration

Your files are cached and delivered through servers closest to your users, ensuring fast loading times for images, videos, and documents for users worldwide. This edge acceleration ensures optimal performance regardless of where your users are located.

## Using Storage in Your Application

You can directly use resource links in your created applications. Simply reference the file URLs provided by superun Storage in your code, and the files will be delivered through the global CDN.

### Viewing Costs

Storage usage and costs can be viewed directly in your billing dashboard. Monitor your storage usage and costs to keep track of your expenses.

## Current Limitations

<Warning>
  Currently, superun Storage file management is only available through the **Build → Services → superun Storage** interface. File uploads via chat attachments are not yet supported.
</Warning>

For now, you'll need to use the superun Storage interface to upload and manage your files. Chat-based file uploads will be supported in future updates.
