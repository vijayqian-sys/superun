您可以在自定义域上发布 superun 项目,以改善品牌推广和 SEO.

## 为什么使用自定义域

- 一致的品牌看起来更专业
- 更好的搜索索引
- 更容易记住和分享

## 要求

- 拥有一个域名
- 使用 Vercel/Netlify 等平台进行托管和自动证书

## Connect your domain（示例:Vercel）

1. 将您的项目连接到 Vercel
2. 在 Vercel 域下添加域
3. 按照 DNS 说明进行操作（例如,CNAME 为 `cname.Vercel-dns.com`）
4.等待SSL颁发（几分钟到几小时）

## 其他提供商

- Netlify:导入存储库 → 域管理 → 遵循 DNS 指南
- Namecheap + GitHub 页面:在 GitHub 上托管并按照指南配置 A/CNAME

## 常问问题

- 域已连接但无法访问:检查 DNS 传播 (dnschecker),确保仅 DNS 模式,验证颁发的 SSL
- 支持的子域:是 - 在连接流中输入子域
- SSL:托管提供商通常自动处理发行和续订
