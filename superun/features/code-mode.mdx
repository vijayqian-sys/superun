代码模式允许您**直接查看项目代码**.您可以在**构建**标签页中点击切换按钮来切换到代码模式.

> **注意**:代码模式功能（包括查看和下载代码）需要**付费用户**才能使用.免费用户无法访问代码模式.

## 如何使用代码模式

1. 在项目中导航到**构建**标签页
2. 点击**代码模式**切换按钮,从视觉视图切换到代码视图
3. 浏览和检查项目代码结构
4. 如需下载代码,点击下载按钮（需要付费用户权限）

代码模式提供只读的代码视图,让您轻松了解项目的结构和实现方式.
