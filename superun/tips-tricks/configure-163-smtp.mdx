## 为什么推荐用网易个人邮箱?

很多人一开始会被「企业邮箱,邮件服务供应商」吓到,其实如果只是做产品验证,内部通知,**用个人网易邮箱就足够**:

- **无需企业资质**:只要手机号就能注册 163 邮箱.
- **5 分钟开通 SMTP**:在设定里开启 IMAP/SMTP,生成授权码即可.
- **能直接接入 superun**:把 SMTP 配置填进环境变数,就能从工作流,动作里发邮件.

下面以「网易 163 邮箱」为例,带你一步步完成配置.

---

## 步骤一:注册或准备一个 163 邮箱

1. 打开 `https://mail.163.com/`.
2. 使用手机号注册一个新帐号,例如 `myname@163.com`.  
3. 完成后,你会得到:
   - 邮箱地址:例如 `myname@163.com`
   - 登录密码:用来登录网页邮箱  

> 注意:**登录密码 ≠ 之后配置 SMTP 用的「授权码」**,兩者是不和的.

---

## 步骤二:在网易邮箱里开启 IMAP/SMTP 并生成授权码

1. 使用浏览器登录你的 163 邮箱（`mail.163.com`）.
2. 在页面顶部黄色导航栏中,找到你邮箱地址右侧的 **「设定」** 按钮.
3. 点击「设定」后,会展开一个下拉选单,从选单中点击 **「POP3/SMTP/IMAP」**.
4. 进入该页面后,在「IMAP/SMTP 服务」一栏:
   - 如果尚未开启,直接勾选或点击 **「开启」**.

### 已开启但发信失败?建议重启服务

如果你确认已开启过,但在 superun 内发信一直报错,可以按以下方式重新生成授权码:

1. 在「IMAP/SMTP 服务」右侧先点 **「关闭」**,确认关闭.
2. 关闭后,再点一次 **「开启」**.
3. 系统会要求进行身份验证:
   - 通常是向你绑定的手机号发送验证简讯;
   - 输入简讯里的验证码,完成验证.
4. 验证通过后,页面会显示一组 **「授权码」**:
   - 是一串随机字串（例如 `ab3XD9kLm...`）;
   - **请立刻复制并妥善保存**,后面配置 SMTP 就要用它.

> 再提醒一次:
> - 授权码只用于 IMAP/SMTP 登录.
> - 在应用程序里填「密码」时,要填 **授权码**,不是你登录邮箱时用的密码.

---

## 步骤三:确认 SMTP 服务器参数

在和一个「POP3/SMTP/IMAP」设定页面底部,你会看到官方给出的 SMTP 资讯,大致如下:

- **SMTP 服务器**:`smtp.163.com`
- **端口（Port）**:`465`（使用 SSL 加密）
- **发信帐号**:你的网易邮箱,例如 `myname@163.com`
- **发信密码**:剛剛生成的授权码

只要这幾项资讯确认无误,就可以切回 superun 开始填环境变数了.

---

## 步骤四:在 superun 中填写 SMTP 环境变数

在 superun 里新增环境变数时,你会看到如下图所示的表单栏位:

![在 superun 中新增 SMTP 环境变数的界面](https://b.ux-cdn.com/uxarts/files/t20260106162112/i0b9uwys.png)

请按下面方式填写（以 163 邮箱为例）:

- **`SMTP_HOST`**:
  填写网易 SMTP 服务器地址
  → `smtp.163.com`

- **`SMTP_PORT`**:
  填写 SSL 加密端口
  → `465`  
  如果 superun 或你的执行环境有「是否使用 SSL/TLS」选项,记得勾选.

- **`SMTP_USER`**:
  填写你的网易邮箱帐号
  → 例如 `myname@163.com`

- **`SMTP_PASS`**:
  填写刚才生成的 **授权码**（不是登录密码）

- **`SMTP_FROM_NAME`**:
  填写收件人看到的「发件人名称」,可以自订,例如:
  → `我的通知`,`系统讯息`,`运营小助手` 等.

全部填好后,点击 **「提交」** 即可.

---

## 步骤五:发送测试邮件

1. 在 superun 中建立一个最简单的流程或动作:  
   - 例如触发后呼叫「发送邮件」动作,  
   - 收件人先填你自己的另一个邮箱,用来测试.
2. 存储并执行这个流程,观察是否成功:
   - 如果执行成功,幾秒内你就能在收件箱收到邮件.
   - 如果报错,先回头检查 SMTP 设定是否有打错字.

## 常见问题排查

### 1. 一直提示「认证失败」?

- 检查 `SMTP_USER` 是否为完整邮箱地址（一定要包含 `@163.com`）.
- 检查 `SMTP_PASS` 是否用的是 **最新的授权码**.
- 确认「POP3/SMTP/IMAP」页面里,IMAP/SMTP 服务确实是 **已开启** 状态.

### 2. 端口一定要 465 吗?

- 官方推荐使用 `465 + SSL`,安全性与稳定性最好.
- 极少数环境如果无法连线 465,可以尝试 `25` 或 `587`,但需同时调整程序与防火墙设定.

### 3. 126,yeah 等其它网易系邮箱可以用吗?

可以,只要:

- 在对应邮箱的设定里和样开启 IMAP/SMTP,生成授权码;
- 把服务器地址改成对应的,例如:
  - `smtp.126.com`
  - `smtp.yeah.net`
- 其它参数（端口,使用授权码）与 163 基本一致.

### 4. 邮件标题或内容出现乱码（如 `=20` 等字符）?

邮件乱码通常是由以下原因引起的:

- 使用了 div 布局,某些邮件客户端不兼容
- SMTP 内容编码设置不正确
- HTML 结构过于复杂

**解决方案**:完全重写邮件模板,采用最兼容的方式:

- **表格布局**:使用 `<table>` 替代 `<div>`,所有邮件客户端都支持
- **标准 XHTML**:使用正确的 DOCTYPE 声明
- **内联样式**:所有 CSS 都写在 HTML 标签内
- **UTF-8 编码**:正确设置字符编码
- **简化结构**:移除复杂嵌套,避免渲染问题

---

## 一句话總结

**用手机注册一个 163 邮箱 → 在设定里开启 IMAP/SMTP 并拿到授权码 → 在 superun 的 SMTP 环境变数中填入 `smtp.163.com / 465 / 邮箱地址 / 授权码 / 发件人名稱`,就能用个人邮箱在 superun 里稳定地自动发送邮件.**
