## 两种方式

### 1. 一次性生成
一次性生成完整页面:

```
You are a CRO expert. Build a landing page for [your product/audience], following conversion best practices.
```

### 2. 逐节
逐节构建:主视觉,定价,常见问题解答,图库……
1) 生成一个空白主视觉 → 2) 使用可编辑设计来调整副本/color/布局 → 3) 添加图像/动画/定价 → 4) 对齐主题（深色/玻璃/霓虹灯等）

## 布局灵感和工具
- Mobbin:真实例子
- 21st.dev:动画/组件
-Remove.bg:背景去除
- 复制:AI图标和插图

## 进阶功能
- P5.js自定义效果（粒子/视差/星空）
- Resend 用于联系和自动回复
- Stripe 付款链接
- Supabase 用于表单存储和滥用保护

## 登陆页面 101
- 单一主要目标（呼叫/购买/候补）
- 清晰的标题和社会证明
- 布局符合用户期望

## 静态与动态
默认静态;使用表单,Supabase,Stripe 等添加动态功能.
