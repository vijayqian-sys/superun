## 快速流动
1. 在superun中新建一个空白项目
2. 连接 Supabase （项目 URL / API 密钥）
3. 将需求分解为小块,通过聊天生成计划,确认,然后执行
4. 使用可编辑设计来调整 UI 和响应能力
5. 发布并运行回归检查

## Supabase 基础知识
- 组织和项目 → 获取 URL 和密钥
- 启用电子邮件登录（您可以最初禁用验证电子邮件）
- 设计表模式和RLS策略

## 典型页面
- 仪表板（Columns表/统计）
- 编辑器（标题/标签/内容区域）
- 设置（配置文件和 API 配置）

## 调试技巧
- 先调查,后修复;如果需要,回滚到工作版本
- 使用版本差异来定位更改和回归
