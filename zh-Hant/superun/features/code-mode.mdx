代碼模式允許您**直接查看專案代碼**.您可以在**構建**標籤頁中點擊切換按鈕來切換到代碼模式.

> **注意**:代碼模式功能（包括查看和下載代碼）需要**付費用戶**才能使用.免費用戶無法訪問代碼模式.

## 如何使用代碼模式

1. 在專案中導航到**構建**標籤頁
2. 點擊**代碼模式**切換按鈕,從視覺視圖切換到代碼視圖
3. 瀏覽和檢查專案代碼結構
4. 如需下載代碼,點擊下載按鈕（需要付費用戶權限）

代碼模式提供只讀的代碼視圖,讓您輕鬆了解專案的結構和實現方式.
