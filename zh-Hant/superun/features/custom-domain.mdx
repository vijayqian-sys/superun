您可以在自定義域上發布 superun 項目,以改善品牌推廣和 SEO.

## 為什麼使用自定義域

- 一致的品牌看起來更專業
- 更好的搜索索引
- 更容易記住和分享

## 要求

- 擁有一個域名
- 使用 Vercel/Netlify 等平台進行託管和自動證書

## Connect your domain（示例:Vercel）

1. 將您的項目連接到 Vercel
2. 在 Vercel 域下添加域
3. 按照 DNS 說明進行操作（例如,CNAME 為 `cname.Vercel-dns.com`）
4.等待SSL頒發（幾分鐘到幾小時）

## 其他提供商

- Netlify:導入存儲庫 → 域管理 → 遵循 DNS 指南
- Namecheap + GitHub 頁面:在 GitHub 上託管並按照指南配置 A/CNAME

## 常問問題

- 域已連接但無法訪問:檢查 DNS 傳播 (dnschecker),確保僅 DNS 模式,驗證頒發的 SSL
- 支持的子域:是 - 在連接流中輸入子域
- SSL:託管提供商通常自動處理髮行和續訂
