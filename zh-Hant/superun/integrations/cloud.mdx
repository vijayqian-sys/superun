superun 雲服務 是一個全棧雲平台,負責處理應用程序的每個部分 - 從用戶看到的前端到在幕後為其提供支持的後端.您所要做的就是直接在 superun 中描述您想要的內容,superun 會自動生成前端和後端,準備在雲中運行.您無需管理硬件或複雜的基礎設施,而是可以按需訪問託管和計算,並隨著應用程序的增長而自動擴展. superun 雲服務 利用 Supabase 的開源基礎,這意味著您從第一天起就獲得了一個生產就緒的環境,而無需單獨設置 Supabase.您可以使用生產就緒的數據庫,實時更新和用戶身份驗證,無需任何手動設置步驟.

## 為什麼選擇superun 雲服務

* **隨著您的成長而擴展**:從您的第一個原型到數百萬用戶,superun 可根據您的需求無縫擴展.
* **默認安全**:企業級身份驗證,細粒度訪問控制和合規性最佳實踐可保護您的應用程序和數據.

## 啟用 superun 雲服務

默認情況下,您的工作區啟用 superun 雲服務.您可以在用戶首選項中管理項目的 superun 雲服務 行為:**構建 → 服務 → superun 雲服務. **

### 用戶偏好

雲集成的默認設置是**每次詢問**,這意味著您決定是否為每個項目啟用或禁用雲.您可以隨時從 **構建 → 服務 → superun 雲服務 更改您的首選項. ** 當某個功能需要後端功能（例如身份驗證）時,superun 會提示您.您有三個選擇:
* **允許**:啟用當前項目的集成.
* **拒絕**:拒絕此請求的集成（稍後可能會再次詢問您）.
* **調整首選項**:更改未來項目的默認行為（不影響當前項目）.

## 配置 superun 雲服務 工具

用戶首選項中的每個雲工具都有三個選項:
* **始終允許**:superun 自動執行操作,無需請求審核或批准.
* **每次都詢問**:每當需要執行操作時,superun 都會請求您的批准.
* **絕不允許**:superun 阻止該操作,通知您需要雲訪問,並指示您啟用雲.

您可以隨時從 **構建 → 服務 → superun 雲服務 更改您的雲工具首選項. **

## superun 雲服務 功能

通過單擊 **預覽** **模式** 旁邊的"+"按鈕,您可以在任何 superun 項目中找到 **雲選項卡**. superun 雲服務 圖標 以下功能和設置位於"雲"選項卡內:

### 資料庫

* **自動生成架構**:用簡單的語言描述您需要的數據,superun 會為您創建表和架構.
* **無需 SQL 進行管理**:直接在 UI 中查看,編輯和組織記錄.更新值,檢查表並保持應用程序的數據同步,而無需編寫查詢.

### 用戶和身份驗證

* **用戶管理**:查看用戶帳戶,跟踪活動並管理訪問控制.
* **內置身份驗證**:superun 可以為您的應用程序生成註冊/登錄頁面.支持電子郵件,電話和 Google 登錄.
* **默認安全**:身份驗證由 superun 雲服務 支持,確保用戶憑據安全.

#### 開啟身份驗證

1. 進入 **研發 → 服務 → superun 雲服務** 詳情頁面
2. 點擊「**身份驗證設定**」
3. 在「**登入方式**」中可以選擇：
   - **電子郵件**: 啟用郵箱密碼登錄
   - **電話**: 啟用手機驗證碼登錄
   - **Google**: 啟用 Google 登錄

啟用後,superun 會自動為您的應用程序生成對應的註冊/登錄頁面.

### 人工智慧

* **內置 AI 功能**:superun 雲服務 具有內置 AI,因此您可以快速創建功能,使您的應用程序更智能,更具吸引力.有關詳細信息,請參閱superun AI.
* AI 支持的功能的一些示例包括:
* **人工智能摘要** – 自動將長文本壓縮為清晰的要點
* **人工智能聊天機器人或代理** – 在您的應用程序中構建對話助手
* **情緒檢測** – 大規模了解用戶反饋
* 您可以從 AI 概覽中監控項目的 superun AI 使用情況和性能.

### 紀錄

* **實時監控**:實時觀察應用程序活動.
* **調試和跟踪問題**:識別錯誤,跟踪事件並確保您的後端正常運行.

## 示例流程 - 提示 superun 構建 CRM 工具

以下是如何使用 superun 雲服務 構建 CRM 工具的分步示例:

### 第 1 步:組織數據（數據庫）

首先描述您需要的數據:

```
I want to build a CRM tool. I need to store:
- Companies (name, industry, website, address)
- Contacts (name, email, phone, company they work for)
- Deals (title, value, stage, contact, company)
- Notes (content, related to deals or contacts)
```

superun 自動為您創建數據庫架構和表.

### 第 2 步:確保安全（用戶和身份驗證）

要求 superun 添加身份驗證:

```
Add user authentication. Only authenticated users should be able to access the CRM.
```

superun 配置身份驗證並為您的應用程序生成註冊/登錄頁面.

### 第三步:添加人工智能超能力（AI）

使用 AI 增強您的應用程序:

```
Add AI-powered summaries for deal notes. When a user writes a note, automatically generate a summary.
```

superun 集成了 AI 功能,可使用內置 AI 自動生成摘要.
