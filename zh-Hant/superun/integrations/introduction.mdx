superun 具有原生無縫集成,可以導出代碼並添加後端.但如果您想為應用程序添加特定功能,也可以集成外部 API.

## 什麼是 API?

API 是軟件應用程序相互通信和共享數據的一種方式.不同的 API 提供不同的功能,例如發送電子郵件或處理付款.您可以在我們博客上的這篇文章中詳細了解 API 的強大功能.
