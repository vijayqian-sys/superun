# 開箱即用 API 密鑰

我們提供開箱即用 API 密鑰,您可以使用它們來測試您的整合.這些密鑰讓您無需設定自己的 API 憑證即可快速開始.

<Warning>
  **重要**:這些 API 密鑰在所有使用者之間共享,僅用於測試目的.它們可能因以下原因而無法使用:
  
  - 多個使用者的高使用量
  - 提供商的速率限制或限制
  - 提供商政策的變更
  
  對於生產環境使用,我們強烈建議使用您自己的 API 密鑰.
</Warning>

## 可用的 API 密鑰

- **[高德地圖](/zh-Hant/superun/ready-to-use-keys/amap)** - 使用高德地圖服務實現地圖,地理編碼和基於位置的功能.
- **[Unsplash](/zh-Hant/superun/ready-to-use-keys/unsplash)** - 整合 Unsplash API 以取得高品質,免版稅的圖片和照片搜尋功能.
- **[Pexels](/zh-Hant/superun/ready-to-use-keys/pexels)** - 整合 Pexels API 以取得高品質,免版稅的圖片和照片搜尋功能.
- **[Windy Weather(Windy.com)](/zh-Hant/superun/ready-to-use-keys/windy)** - 整合 Windy API 以取得即時天氣資料,預報和氣象視覺化.
