## 兩種方式

### 1. 一次性生成
一次性生成完整頁面:

```
You are a CRO expert. Build a landing page for [your product/audience], following conversion best practices.
```

### 2. 逐節
逐節構建:英雄,定價,常見問題解答,圖庫……
1) 生成一個空白英雄 → 2) 使用可編輯設計來調整副本/color/佈局 → 3) 添加圖像/動畫/定價 → 4) 對齊主題（深色/玻璃/霓虹燈等）

## 佈局靈感和工具
- Mobbin:真實例子
- 21st.dev:動畫/組件
-Remove.bg:背景去除
- 複製:AI圖標和插圖

## 進階功能
- P5.js自定義效果（粒子/視差/星空）
- Resend 用於聯繫和自動回复
- Stripe 付款鏈接
- Supabase 用於表單存儲和濫用保護

## 登陸頁面 101
- 單一主要目標（呼叫/購買/候補）
- 清晰的標題和社會證明
- 佈局符合用戶期望

## 靜態與動態
默認靜態;使用表單,Supabase,Stripe 等添加動態功能.
