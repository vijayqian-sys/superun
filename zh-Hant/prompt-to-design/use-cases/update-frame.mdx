title: "更新或改造現有設計"
---

## 場景
您想要對舊的 UI 進行現代化改造,但保留其結構.

## 挑戰
手動重構 UI 速度慢且容易出錯.

## 如何讓 prompt.to.design 提供協助

- 選擇您想要刷新的舊 **Prompt.to.design 框架**
- 輸入現代化提示,例如:
**“使此設置頁面現代化,同時保持相同的佈局結構.使用更清晰的排版,更新的表單組件,改進的間距和更精美的側邊欄設計.”**
- 單擊**生成**以應用煥然一新的現代風格

<img
  src="/images/update-oldframe/1.png"
  alt="Select old frame and apply modernization prompt"
  style={{ marginTop: '1rem', borderRadius: '8px' }}
/>

- 生成開始.右側面板顯示AI對舊版佈局的分析結果及推薦的風格改進方案.

<img
  src="/images/update-oldframe/2.png"
  alt="Modernization analysis"
  style={{ marginTop: '1rem', borderRadius: '8px' }}
/>

- 在預覽中查看更新的設計,如有必要進行完善,或**導入到 Figma**.
