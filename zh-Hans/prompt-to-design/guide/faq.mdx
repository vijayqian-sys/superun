## 入门
<AccordionGroup>
  <Accordion title="什么是Prompt.to.design?">
AI 了解您的style,一键创建,编辑和原型.
  </Accordion>

<Accordion title="我可以使用 Prompt.to.design 构建什么?">
    登陆页面,仪表板,产品 UI 和营销布局 - 在 Figma 中完全可编辑.
  </Accordion>

<Accordion title="我需要设计经验才能使用它吗?">
不会.描述一下您的想法,插件就会创建一个干净,可编辑的布局.
  </Accordion>

<Accordion title="我该如何开始?">
    从 Figma Community 安装 - 打开插件 - 点击 Kick Start - 输入您的提示.
  </Accordion>
</AccordionGroup>

## 特点和用途
<AccordionGroup>
  <Accordion title="核心特点是什么?">
Kick Start,样式转移,区域设计,智能填充和修改元素.
  </Accordion>

  <Accordion title="它可以与 Figma 组件一起使用吗?">
是的.所有输出都使用Frame,自动布局和兼容组件.
  </Accordion>

<Accordion title="我可以上传自己的参考资料吗?">
是的.支持图像,PDF 和屏幕截图作为视觉指南.
  </Accordion>

<Accordion title="我可以运行幾代?">
免费试用:20/天. Plus 计划:100 个每月积分.
  </Accordion>
</AccordionGroup>
