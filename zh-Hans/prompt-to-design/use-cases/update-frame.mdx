title: "更新或改造现有设计"
---

## 场景
您想要对旧的 UI 进行现代化改造,但保留其结构.

## 挑战
手动重构 UI 速度慢且容易出错.

## 如何让 prompt.to.design 提供协助

- 选择您想要刷新的旧 **Prompt.to.design 框架**
- 输入现代化提示,例如:
**“使此设置页面现代化,同时保持相和的布局结构.使用更清晰的排版,更新的表单组件,改进的间距和更精美的侧边栏设计.”**
- 单击**生成**以应用焕然一新的现代风格

<img
  src="/images/update-oldframe/1.png"
  alt="Select old frame and apply modernization prompt"
  style={{ marginTop: '1rem', borderRadius: '8px' }}
/>

- 生成开始.右側面板显示AI对旧版布局的分析结果及推荐的风格改进方案.

<img
  src="/images/update-oldframe/2.png"
  alt="Modernization analysis"
  style={{ marginTop: '1rem', borderRadius: '8px' }}
/>

- 在预览中查看更新的设计,如有必要进行完善,或**导入到 Figma**.
