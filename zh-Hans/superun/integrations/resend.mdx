## 概述

Resend 是专为开发人员设计的现代电子邮件 API.它提供了一种简单,可靠的方法来从 superun 应用程序发送交易和营销电子邮件.

## 什么是 Resend

Resend 提供开发人员友好的电子邮件服务:

- **简单的 API:** 干净,RESTful API,易于集成
- **高送达率:** 针对收件箱放置进行了优化
- **React Email:** 对基于 React 的电子邮件模板的内置支持
- **Webhooks:** 实时交付和参与跟踪
- **分析:** 详细的电子邮件绩效指标
- **模板:** 适用于常见用例的预构建电子邮件模板

## 主要优点

- **开发者体验:** 由开发者设计,为开发者服务
- **高送达率:** 比传统 SMTP 更好的收件箱放置
- **React 集成:** 对 React 电子邮件组件的本机支持
- **实时跟踪:** 用于交付和参与活动的 Webhooks
- **可扩展:** 处理大量电子邮件发送
- **价格实惠:** 有竞争力的价格和慷的的免费套餐

## 入门

### 1. 创建一个 Resend 帐户

1. 前往 [resend.com](https://resend.com) 并注册
2. 验证您的电子邮件地址
3. 完成帐户设置过程

### 2. 获取您的 API 密钥

1. 前往 Resend 仪表板中的 **API Keys**
2. 点击**创建 API 密钥**
3. 为其命名（例如,"superun App"）
4. 复制 API 密钥（以 `re_` 开头）
5. 确保其安全,切勿在客户端代码中公开它

### 3. 在 superun 对话中配置 Resend

当您需要在 superun 项目中使用 Resend 时,只需在对话中提及即可.superun 会自动提示您输入 Resend API 密钥.只需贴上从 Resend 复制的 API 密钥,superun 会自动处理其余的配置.
