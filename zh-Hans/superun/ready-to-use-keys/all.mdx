# 开箱即用 API 密钥

我们提供开箱即用 API 密钥,您可以使用它们来测试您的整合.这些密钥让您无需设定自己的 API 凭证即可快速开始.

<Warning>
  **重要**:这些 API 密钥在所有用户之间共享,仅用于测试目的.它们可能因以下原因而无法使用:
  
  - 多个用户的高使用量
  - 提供商的速率限制或限制
  - 提供商政策的变更
  
  对于生产环境使用,我们强烈建议使用您自己的 API 密钥.
</Warning>

## 可用的 API 密钥

- **[高德地图](/zh-Hans/superun/ready-to-use-keys/amap)** - 使用高德地图服务实现地图,地理编码和基于位置的功能.
- **[Unsplash](/zh-Hans/superun/ready-to-use-keys/unsplash)** - 整合 Unsplash API 以取得高品质,免版税的图片和照片搜寻功能.
- **[Pexels](/zh-Hans/superun/ready-to-use-keys/pexels)** - 整合 Pexels API 以取得高品质,免版税的图片和照片搜寻功能.
- **[Windy Weather(Windy.com)](/zh-Hans/superun/ready-to-use-keys/windy)** - 整合 Windy API 以取得即时天气资料,预报和气象视觉化.
