---
title: "Recreate a design style from a reference"
---


## Scenario
You already have a beautifully designed page and want to expand more screens with the same visual style.

## Challenge
Manually keeping visual consistency across new pages takes a lot of time and effort.

## How prompt.to.design helps

- Upload a reference image or select an existing **Figma Frame**  
- Enter your prompt and choose the target device size  
- Click **Generate** to let the plugin analyze the reference layout, spacing, colors, and style rules

<img
  src="/images/design-from-reference/1.png"
  alt="Reference setup and prompt input"
  style={{ marginTop: '1rem', borderRadius: '8px' }}
/>

- Generation begins. The right panel will show prompt.to.design's analysis of the reference style and your request.

<img
  src="/images/design-from-reference/2.png"
  alt="AI analysis screen"
  style={{ marginTop: '1rem', borderRadius: '8px' }}
/>

- Review the generated result, refine if needed, or **Import to Figma** to continue your work.

<img
  src="/images/design-from-reference/3.png"
  alt="Preview generated design"
  style={{ marginTop: '1rem', borderRadius: '8px' }}
/>
